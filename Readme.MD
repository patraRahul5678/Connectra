<h1 align="center">✨ Connectra ✨</h1>

Connectra is a scalable real-time communication and language exchange platform featuring secure JWT authentication, live messaging with typing indicators and reactions, and 1-on-1 & group video calls with screen sharing and recording. Built using React, Express, MongoDB, TailwindCSS, TanStack Query, Zustand, and Stream, it offers 32 unique UI themes, robust error handling, and free cloud deployment for a seamless user experience.

## 📖 About the Project

Connectra is a feature-rich communication platform designed to connect people across the world through language exchange and real-time interactions.

The project was built to gain hands-on experience with production-grade concepts such as real-time messaging, video calling, state management, authentication, and scalable application design.

Users can communicate through instant messaging, participate in video calls, share screens, react to messages, and personalize their experience with multiple UI themes.

## 🧪 Environment Variables

### Backend (`/backend`)

```env
PORT=5001
MONGO_URI=your_mongo_uri
STREAM_API_KEY=your_stream_api_key
STREAM_API_SECRET=your_stream_api_secret
JWT_SECRET_KEY=your_jwt_secret
NODE_ENV=development
```

### Frontend (`/frontend`)

```env
VITE_STREAM_API_KEY=your_stream_api_key
```

## ⚙️ Installation & Setup

### Clone the Repository

